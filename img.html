Image Tag

  • The img tag in HTML is used to embed images
  • Images from local and the internet can be used and displayed on our website using img tag.
  • When using the image tag, it's common to place the image source inside the src attribute and can place a text that might shown alt text is displayed if the image cannot be loaded or when a user hovers over the image.

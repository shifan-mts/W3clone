Title and Text tags

Title tag

  • The title tag in HTML is placed within the head section and defines the title of the webpage displayed on the browser tab or window.

Text tags

  • Text tags like p (paragraph) in HTML structure and format textual content.
  • p and h tags are the basic text tags in HTML, in which h tags automatically ends with the break
  • h tag ranges from h1 to h6 , the number specifies the size of the text.
  • if you code like h7 tags , then the size of the text remains same as h6 tags
